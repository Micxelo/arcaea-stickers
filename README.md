# Arcaea Stickers —— 生成你的 Arcaea 贴纸

[![MIT License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

## 🎮 介绍

SEKAI 贴纸工具那么多，难道 Arcaea 就没有吗？

于是，我们制作了 Arcaea Stickers，一个专门为 Arcaea 玩家设计的贴纸生成工具，支持一键生成 Link Play 表情样式的贴纸，满足你在各类社交平台的分享需求！

## ✨ 功能

- [x] 选取角色和自定义文字
- [x] 一键下载/复制贴纸
- [x] 文本的分色绘制
- [x] 更好的曲线文字支持
- [x] 操作历史（撤销/重做）
- [ ] 贴纸制作次数统计
- [ ] 在社交平台直接分享

## 🚀 快速开始

### 前端要求

- Node.js 18+
- npm 11+

```bash
git clone https://github.com/Micxelo/arcaea-stickers.git
cd arcaea-stickers
npm install
npm run dev # 访问 localhost:3000 查看，支持热更新

npm run build # 构建生产版本
```
